
# 快速开始

## 准备工作

仅需两步快速安装 RiskScanner：

1.  准备一台不小于 4 G内存的 64位 Linux 主机；
2.  以 root 用户执行如下命令一键安装 RiskScanner。

!!! info "部署服务器推荐"
    * 操作系统: CentOS 7.x
    * CPU/内存: 2核4G及以上
    * 磁盘空间: 100G

## 快速安装

```sh
curl -sSL https://github.com/riskscanner/riskscanner/releases/latest/download/quick_start.sh | sh
```
